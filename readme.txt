TVGuide addon support is through .strm files
Format of a .strm file is:
plugin://plugin.video.ustvnow/?name=ABC&mode=play

Where name is the channel name given by USTVNow in exactly the same case as given in the menu.
&mode=play tells the addon to play the channel

see the .strm files in the plugin.video.ustvnowtvguide directory for examples
